Les directives

Les directives de composants

ngStyle

La directive ngStyle permet de manipuler le CSS de votre élément HTML à l'aide d'un objet de style.

Exemples :

[ngStyle] [style]
Attention, un import est nécessaire. Pas d'import et comportement identique.

[ngClass]

Cette directive permet de gérer la présence ou non de class CSS à l'aide de valeur logique.

Exemple :

[ngClass] [class]
Attention, un import est nécessaire. Pas d'import et comportement identique.

Directives Structurelles

*ngIf → {{'@if'}}

La directive if permet l'affichage ou le masquage d'élément HTML selon des booléens du TypeScript.
